
# Ficha del lenguaje de programación ```Nim```
## Propósito
> *General*.
---
## Paradigma
> *Multiparadigma: compilado, corriente, procedimental, imperativo, funcional, orientado a objetos y metaprogramación.*
---
## Fortalezas y debilidades
### Fortalezas
> * *Rendimiento de un lenguaje compilado (C) a la par de una facilidad de lectura y productividad de un lenguaje como Python.*
> * *Tiene una curva de aprendizaje menos pronunciada que C o C++.*
> * *Contiene funciones puras.*
> * *Admite compilación en Javascript.*
> * *Tiempo de ejecución veloz.*
> * *Tiene soporte para macros*
### Debilidades
> * *No agrega innovaciones notorias frente a otros lenguajes de programación como Rust.*
> * *Dada su baja popularidad, carece de referencias variadas.*
> * *Tiene un tipado estricto.*
> * *Carece de documentación o suficientes proyectos de referencia.*
---

---
## Runtime
